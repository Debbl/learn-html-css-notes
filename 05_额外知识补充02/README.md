# link 元素

> link 元素是外部资源链接元素，规范了文档与外部资源的关系

- link 元素通常在 head 元素中
- 链接 CSS
- 创建 站点图标
- 属性
  - href
  - ref https://developer.mozilla.org/zh-CN/docs/Web/HTML/Link_types
    - icon
    - stylesheet

# 认识进制

- 二进制 `0b` binary
- 八进制 `0o` Octonary
- 十六进制 `0x` hexadecimal
